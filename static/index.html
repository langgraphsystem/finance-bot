Smart OCR чеков

Умное извлечение данных. Отправьте фото скомканного чека. AI мгновенно распознает мерчанта, товары, налоги и раскидает по категориям с помощью Gemini Flash 3.

receipt_parser.py
import finance_ai

receipt = finance_ai.scan("photo.jpg")
print(receipt.merchant) # -> "Shell"
print(receipt.total)    # -> 150.00
