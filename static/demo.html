Smart OCR чеков

Отправьте фото скомканного чека. AI мгновенно распознает мерчанта, товары, налоги и раскидает по категориям через Gemini 3 Flash.

receipt_parser.py
import finance_ai

receipt = finance_ai.scan("photo.jpg")
print(receipt.merchant) # -> "Shell"
print(receipt.total)    # -> 150.00
